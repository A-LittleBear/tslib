C library for filtering touchscreen events

Supports...

tslib is cross-platform. On Linux and FreeBSD it supports all modern devices using input event drivers and includes multitouch support.

Furthermore it supports the Palm Pre's CY8MRLN, UCB1x00, Sharp Zaurus sl-c7x0 and sl-5500/5000d, HP iPaq H3600 and DMC, Hitachi Webpad, IBM Arctic II, Tatung Webpad, eGalax 0112, 210 and 0100, Waveshare and more, in part for other host platforms too.

What is tslib used for?

tslib is used to access touch screen devices, no matter how old or weird it's driver may be. It is used to optimize the touch user experience at runtime, applying mathematical filters in any number or order. Filters are configured in one simple configuration file. The libts library is highly configurable and portable. tslib includes tools to calibrate, test and use the library. It is used in many commercial products. tslib includes documentation on how to use it.

Who makes tslib?

tslib exists thanks to various contributors from companies across the world and is open to your input. It was originally written by Russel King in 2001 and is actively maintained.
